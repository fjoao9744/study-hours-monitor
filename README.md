# study-hours-monitor
Este projeto vem a partir de uma necessidade minha de registros de horas no dia para organizção, o site conta com 4 partes principais e suas respectivas tecnologias:

## Front-end:
- Tailwind CSS
- JavaScript + Chart.js

## Back-end:
- Django (rotas, lógica, autenticação, estáticos)

### API (parte do Back-end):
- Django REST Framework (comunicação de dados)
- PostgreSQL

## Deploy (Infraestrutura):
- Fly.io (app Django)
- Neon.tech (PostgreSQL hospedado)
